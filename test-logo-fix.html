🚨 问题描述

在主页的加载动画中，LOGO周围出现了一个不应该存在的白色/灰色背景框，而其他页面的加载动画显示正常。

问题根源：

.global-loading-screen .loading-logo-img { background: rgba(255, 255, 255, 0.1); /* 白色半透明背景 */ padding: 8px; /* 内边距 */ backdrop-filter: blur(10px); /* 背景模糊效果 */ }

✅ 修复方案

移除导致白色框的CSS属性，确保LOGO显示清洁透明。

修复前

background: rgba(255, 255, 255, 0.1); padding: 8px; backdrop-filter: blur(10px);

❌ 产生白色背景框

修复后

background: transparent; border: none; /* 移除padding和backdrop-filter */
